# Slack Task Reminder GAS

Google Spreadsheetに記載されたタスクを検知し、未完了のまま指定時刻を過ぎたものについて、各担当者のSlackへ自動でDMリマインドを送信するGoogle Apps Scriptです。

## 機能
- **条件抽出**: ステータスが「完了」以外、かつD列（リマインド時刻）を過ぎているタスクを抽出。
- **リアクション検知（👀）による自動登録**: Slack上でメッセージに `:eyes:` リアクションを付けると、スプレッドシートの該当日時に自動でリマインド時刻をセット（Webhook連携）。
- **リマインド時刻の自動計算**: 
  - 初回: リアクション時間（C列）＋24時間をD列に自動セット。
  - 2回目以降: 通知が完了したタスクのD列を「翌日の朝8:00」に自動更新（毎日継続してリマインドを行うため）。
- **ステータス情報の記録**: Slackへの送信が成功した場合、連携シートのF列に「配信済」を自動記録。
- **Slack API 連携**: `chat.postMessage` で特定のSlackID(A列)へ直接DM（ダイレクトメッセージ）を一括通知。

## スプレッドシートの構成
- **A列**: 担当者のSlackID (例: `U08UYG...`)
- **B列**: メッセージのリンク（URL）
- **C列**: リアクションが押された時間（タスク化時刻）
- **D列**: リマインド時刻（GASによって自動入力・更新されます）
- **E列**: 進行状況（「完了」と入力されているとリマインド対象から外れます）
- **F列**: リマインド状況（通知成功時にGASによって「配信済」と自動入力されます）

## セットアップ手順

### 1. Slack アプリの設定
1. [Slack API](https://api.slack.com/apps) で新しいアプリを作成。
2. **OAuth & Permissions** で `chat:write` および `reactions:read` スコープを追加。
3. **Bot User OAuth Token** (`xoxb-...`) を取得し、ワークスペースにインストール（Reinstall）。

### 2. GASのデプロイ (clasp経由)
このプロジェクトは、Googleが提供するコマンドラインツール `clasp` で管理できる構造になっています。

1. [Node.js](https://nodejs.org/) がインストールされていることを確認する。
2. ターミナルで `npm install -g @google/clasp` を実行し、Claspをインストール。
3. `clasp login` コマンドでGoogleアカウントを認証する。
4. 本ディレクトリ内に連携済みの `.clasp.json` が配置されているため、`clasp push` コマンドでスクリプトをGASへデプロイできます。

### 3. GAS上での設定とトリガー
1. [GASエディタ](https://script.google.com/home) を開き、アップロードされた `src/main.js` ファイルの先頭にある `SLACK_ACCESS_TOKEN` の値を、取得したボットトークンに書き換える。
2. GASエディタの左側メニューから「トリガー（時計アイコン）」を選択し、「トリガーを追加」をクリック。
3. 今回の通知に合わせ、以下の設定でトリガーを作成する:
   - 実行する関数: `remindPendingTasks`
   - イベントのソース: `時間主導型`
   - トリガーのタイプ: `日付ベースのタイマー`
   - 時刻の選択: `午前9時～10時` （許容されるばらつきの範囲内で実行されます）

### 4. WebhookのデプロイとSlackイベント設定
リアクション検知（👀）を有効にするための設定です。

1. **GASをウェブアプリとしてデプロイ**:
   - GASエディタ右上の「デプロイ」＞「新しいデプロイ」を開く。
   - 種類に「ウェブアプリ」を選択。
   - 次のユーザーとして実行: 「自分」、アクセスできるユーザー: 「全員」に設定してデプロイ。
   - 生成された **ウェブアプリのURL** をコピーする。
2. **Slack Event Subscriptionsの設定**:
   - Slack API管理画面で **Event Subscriptions** を有効化する。
   - **Request URL** にコピーしたウェブアプリのURLを貼り付ける（「Verified」と表示されることを確認）。
   - **Subscribe to bot events** で `reaction_added` イベントを追加し、保存する。

## 開発環境
- Google Apps Script (runtime: V8)
- Clasp / Node.js
